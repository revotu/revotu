在命令行中或程序代码中得到python版本号

发表于 2017-09-05 | 阅读次数

在命令行中

在命令行中通过python -V或是python --version得到python版本号。

在程序代码中

在程序代码中，可以通过导入sys模块，用sys.version或是sys.version_info得到python版本号。

或是导入platform模块，用platform.python_version()得到python版本号。
